# 가시제어자 (Visibility modifiers)

```
public > internal > protected > private
```

### ◼️ public
- 코틀린의 기본 가시제어자
- 어디에서든 접근 가능

### ◼️ internal
- 같은 모듈 내 어떤 파일에서든 접근 가능

### ◼️ protected
- 해당 클래스 내에서와 해당 클래스의 자식 클래스는 접근 가능
- top-level 에서 선언 불가

### ◼️ private
- 해당 파일(.kt) 또는 클래스 내에서만 접근 가능
